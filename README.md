# Calculadora de Emissão de Carbono
Projeto final do curso Github Copilot - Código na prática da plataforma DIO (Digital Innovation One)

## Descrição 
Este projeto é uma aplicação web simples que tem como objetivo calcular a quantidade de gás carbônico emitida durante o deslocamento entre dois pontos, considerando diferentes tipos de veículos. 

O projeto foi desenvolvido com auxilio do Github copilot (Assistente de programação com IA), utilizando HTML, CSS e Javascript. A ideia principal do projeto é conscientizar usuários e empresas sobre o impacto ambiental gerado no deslocamento diário entre dois pontos. 

## Funcionalidades

* Seleção de Rotas (Local de sáida -> Local de chegada)
* Seleção do tipo de veículo
* Cálculo estimado de emissão de CO2
* Exibição do resultado na interface

## Tecnologias utilizadas

* HTML5 - Estrutura do projeto
* CSS3 - Estilização do projeto
* Javascript - Lógica, eventos e manipulação do DOM
* GitHub Copilot - Assitência no desenvolvimento do código
